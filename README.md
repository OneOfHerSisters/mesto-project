# Проект: Место

Веб-сервис Mesto — интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

## Выполнен с использованием:

1. Grid-layout
2. Флексбокс-верстка
3. Медиазапросы
4. Методология БЭМ.
5. JavaScript
